# Go gwttr client

[![Go Report Card](https://goreportcard.com/badge/github.com/alrayyes/gwttr)](https://goreportcard.com/report/github.com/alrayyes/gwttr)

A proof of concept client written in Go to access [wttr.in](https://wttr.in/).

## Building

```shell
go build
```

## Usage

```shell
./gwttr
```
